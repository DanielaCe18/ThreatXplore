Who Are We ?

We are 3rd year students in Cybersecurity at ESGI (Ecole Supérieure de Génie Informatique) localised in Paris, 12rd arrondissment. This website has been created as an annual project.

This website is web vulnerability scanner. Its purpose is to offer an open source solution to detect the vulnerabilities present in a website and give solutions to secure it in case of risk.
